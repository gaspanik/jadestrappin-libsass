# Jadestrappin-libsass

Bootstrap 3 Official + Jade template w/ Gulpjs.

## Requirement

Bower.

## Usage

Install some components and initialize.

	$ npm install

Run gulp

	$ npm start

Open [http://localhost:1337](http://localhost:1337) with your favorite browser.
